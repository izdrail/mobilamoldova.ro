### Description

**Mobila Bucatarie Mina 160 Alb/Ferrara C10**

O mobila modulară elegantă și modernă, Set Mobila Bucatarie Mina 160 Alb/Ferrara C10 este ideală pentru personalul care dore să dezida o mică și elegantă mobila. Setul este compus din 8 corpuri modulare, care pot fi poziționate în diverse configurații, de exemplu: chiuveta în dreapta, coloana centrală sau ca stomac.

Mobila este confecționată din pal de 16mm cu aplicație 0,6, care permite ca manerele să fie placute cu grijă. Fronturile sunt disponibile în 3 variante: Alb, Gri Antracit și Ferrara, iar carcasa este cu finisaj Alb.

Setul Mobila Bucatarie Mina 160 Alb/Ferrara C10 este un produs de alta calitate și este disponibil pentru prețurile menționate.

**Parametri principali:**

* Material: pal de 16mm cu aplicație 0,6
* Dimensii: 162 cm x 60 cm x 200 cm (înașinele adjustable)
* Aspecte: alb, gri antracit și Ferrara
* Conținut: 8 corpuri modulare
* Mane: argintii sau negre
