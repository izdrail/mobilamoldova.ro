### Description

**Canapea Iris Crem / Maro**

Canapea Iris Crem / Maro este o tapetă minimalistă și elegantă care se pot pot utiliza în diverse situații. Design-ul simplu și decorat contribuie la un stil modern și de lux.

Canapea are o baterie placută și anatomică, care oferă un suport stabil. Este perfectă pentru cei care valorizează confortul și stilul. Dimensiunile sale sunt ample, cu o extindere de 184 x 118 cm, care oferă un spațiu de dormit ample.

Este disponibilă în o gamă largă de culori, iar nu este recomandat să se cureze cu agenți chimici. Imaginile cu titlu de prezentare ofer o idee clară despre design-ul tapetului.
