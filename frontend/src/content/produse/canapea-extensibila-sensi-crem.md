### Description

**Canapea Extensibila Sensi Crem - Canapea extensibila Sensi Crem**

Canapea Extensibila Sensi Crem este o piesă de mobilier modernă care oferă un plus de confort și raffinament în orice incaptă. Formele rotunjite și eleganți oferă un design elegant, iar cele doua perne si bratele laterale detasabile oferă sustinere corpului exact în poziția dorita.

**Aspecte principale:**

* Lungime: 190 cm
* Latime: 102 cm
* Inaltime total: 85 cm
* Adancime sezut fara perne: 85 cm
* Adancime sezut cu perne: 65 cm

**Material:**

Canapea este tapitată cu stofa Crem, un material robust și durabil.

**Dezvoltare:**

* Extensibila este de tip Doppelliege, care prezintă un sistem de rol pentru partii de spatar.
* Prima perna este fixată pe jos, iar perna secundară este rotunjită și este fixată pe spate.
* Extensia este de 190 cm, o dimensiune suitable pentru majoritatea camerelor.

**Utilizare:**

Canapea este ideală pentru:

* Sferări de zi
* Camere de zi
* Sala de lucru
* Ateliere

**Persoana care luptă cu linișea:**

Canapea Extensibila Sensi Crem este o soluție eficientă și confortabilă pentru a păstra linișea, a sprijini bravara și a oferi un spațiu de relaxare.
